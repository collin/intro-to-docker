This section will cover the basics of using the terminal. If you're already familiar with the terminal, this might not be very engaging for you. But we need to bring everybody up to a common baseline before we can move on to more advanced topics.

If you're new to the terminal, it might be useful to break out a paper and pen to create a little cheat sheet for common operations, as you'll be running them frequently throughout the following labs.
